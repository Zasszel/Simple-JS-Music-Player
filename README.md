# Simple-JS-Music-Player

A lightweight, responsive web-based music player built with Vanilla JavaScript. This project features a dynamic progress bar, automated playlist cycling, and a clean UI for an effortless listening experience.

## Features

* **Playlist Management:** Easily swap between songs using an array-based data structure.
* **Real-time Progress:** Interactive fill-bar that updates as the song plays.
* **Seek Functionality:** Click anywhere on the progress bar to skip to that part of the song.
* **Smart Formatting:** Custom time-parsing logic to display `MM:SS` duration and current time.
* **Dynamic UI:** Background covers and artist details update instantly upon song change.
* **Auto-Play:** Automatically moves to the next track when the current one ends.
